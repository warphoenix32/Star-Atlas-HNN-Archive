## Preserved Official Text

Ship components are modular parts that determine a ship’s performance, specialization, and customization potential in Star Atlas. Each ship contains specific component slots, filled by default parts that enable core functionality. Over time, these components will be swappable, upgradable and craftable, allowing players to fully configure their ships to match gameplay needs.
Component Overview
Every ship hull includes Tier 0 core components by default, allowing the ship to function. These cannot be removed or traded but can be overwritten by higher-tier components in the future.
Common component types include:
Radar
Warp Drive
Impulse Engine
Power Supply
Shield Generator
Weapon Slot
Missile Bay
Countermeasure Bay
Tractor Beam
Maneuvering Thruster
Hull Plating
Additional modules and weapons will also be equippable, offering specialized functionality for combat, logistics, or exploration. These are separate from the default components.
Legacy Ship Parts Bundles (Airdrop)
On August 8, 2024, a snapshot was taken of all wallets holding Star Atlas ships. As a reward for early adoption, players received Ship Parts Bundles corresponding to each ship owned
These bundles contain higher-tier components and weapons, tied to the specific ship model
They are tradable on the Marketplace today but not yet usable in-game
Once the Ship Configuration Protocol is live, these bundles will be convertible into individual, equipable components
Ships purchased after this snapshot are considered ship hulls only and do not include any of the legacy airdropped components, crew, or weapons.
What Comes With a Ship Today?
Ships bought on the primary or secondary marketplace now include:
✅ Tier 0 core components (non-tradeable, cannot be removed)✅ Utility Rigs (pre-installed, vary by ship purpose)✅ Interior Elements (crew quarters, command deck, etc.)
❌ No weapons❌ No modules❌ No crew❌ No Ship Parts Bundles (if bought after the August 8, 2024 snapshot)
The Future: Ship Configuration Protocol
Once implemented, the Ship Configuration Protocol will:
Allow you to open Ship Parts Bundles
Equip, upgrade, and swap components
Customize ship performance for various game loops like combat, exploration, crafting, and racing
Introduce a player-driven economy of component manufacturing and trade
This system will evolve ship NFTs from semi-fungible tokens into fully customized and player-defined digital assets.
