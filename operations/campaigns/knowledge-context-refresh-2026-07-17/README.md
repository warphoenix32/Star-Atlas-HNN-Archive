# Knowledge Context Refresh — 2026-07-17

This campaign promotes reviewed evidence already present on `main` into human-first knowledge pages. No page relies on C1 or C2 evidence alone; the Medium source-family profile retains C1 limits because article text is not yet preserved. It does not rewrite archive evidence or modify `graph/` or `publication/`.

The campaign preserves these distinctions: proposal versus passage, passage versus payment, plan versus release, test versus mainnet, and official resharing versus first-party authorship. Inferences are included only when their evidence basis and limitations are explicit.

Run `python build_campaign.py`, then `python validate_campaign.py` from this directory. Generated packets and reports are deterministic.
