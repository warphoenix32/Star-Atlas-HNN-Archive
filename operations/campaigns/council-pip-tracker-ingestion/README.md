# Council PIP Tracker Ingestion

This campaign preserves the original and normalized Council workbooks, five sheet exports, 39 operational tracker records, 80 review-guidance records, source schemas, and provenance metadata. It does not convert Council assessments into independently verified fact.

Run `python operations/campaigns/council-pip-tracker-ingestion/build_campaign.py` then `python operations/campaigns/council-pip-tracker-ingestion/validate_campaign.py`.
