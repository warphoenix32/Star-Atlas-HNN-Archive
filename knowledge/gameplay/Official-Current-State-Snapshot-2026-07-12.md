# Official Star Atlas Current-State Snapshot — 2026-07-12

This entry records how Star Atlas publicly described its product ecosystem on 2026-07-12. It is a dated snapshot and must not be silently treated as a timeless description.

## Public positioning

Star Atlas described itself as an AAA space-exploration MMO centered on exploration, territorial competition, combat, crafting, quests, fleet management, and a player-driven economy. The site distinguished third-person gameplay from a planned top-down real-time-strategy fleet view.

## Access and product status

- The website offered early-access keys by email.
- The Unreal Engine client linked to the Epic Games Store.
- SAGE Labs linked to a browser product at `based.staratlas.com`.
- Holosim linked to `holosim.staratlas.com`.
- The Galactic Marketplace linked to `play.staratlas.com`.
- Fleet Command was labeled “Coming soon.”

## Current official product descriptions

### Unreal / core game

The site described ship-to-ship and first-person combat, exploration, crafting, quests, racing, PVE dogfights, ship interiors, and fleet customization. These statements describe current official positioning; feature-level availability requires build-specific verification.

### Fleet Command

The site described a planned switch from third-person gameplay to a top-down view for mass fleet management, territorial control, and faction competition. Because the feature was explicitly marked “Coming soon,” it must remain classified as announced rather than delivered in this snapshot.

### Holosim

Holosim was described as a free-to-play browser simulation offering strategic fleet gameplay without requiring high-end hardware, owned ships, or a crypto wallet.

### SAGE Labs

SAGE Labs was described as a browser economic game supporting fleet management, resource mining, crafting, and SDU scanning.
