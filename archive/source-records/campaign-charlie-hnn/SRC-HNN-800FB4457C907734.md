A list of the economic levers in Star Atlas that affect economic resource distribution.
Star Atlas has touted that it has one of the most robust economic systems in any gaming MMO. Facilitating that dynamic economy are “30 levers” that Star Atlas economists have been quoted saying that they can use to maintain a relative equilibrium in the Star Atlas economy. Until now, what these levers were could be guessed at but were never truly known. That is, until Hologram networked data runners uncovered the mechanisms that these levers control.
Below is a list of the “30 economic levers” that can affect the economy of Star Atlas. Each lever name is in bold with a brief description underneath.
List of Star Atlas Economic Levers
1.) Resources Scalar Multiplication Description: Global lever that scales the quantity of resources
2.) Emission Main BreakerGlobal lever which impacts the distribution of effective emissions
3.) System Richness EmissionsGlobal lever which impacts the effect that system richness has on resource emissions
4.) Ship Size WeightGlobal lever which impacts the effect that ship size has on resource emissions
5.) Resource HardnessGlobal lever which impacts the effect that resource hardness has on resource emissions
6.) Fuel Warp BreakerModule wide lever that directly impacts the nominal cost of warping
7.) Fuel Planet BreakerModule wide lever that directly impacts the nominal cost of planet exit
8.) Fuel Refinement EfficiencyModule wide lever that impacts refinement from R20 (hydrogen) to R4 (fuel)
9.) Mining Food BreakerModule wide lever that directly impacts the nominal cost of mining
10.) Combat Food BreakerModule wide lever that directly impacts the nominal cost of combat
11.) Food Refinement EfficiencyModule wide lever that impacts refinement from R20 (organics) to R4 (food)
12.) Organics Scalar Multiplication Module specific lever that scales the quantity of organics in the economy
13.) Combat Toolkit BreakerModule wide lever that directly impacts the nominal cost of repairing during combat
14.) Ammo Refinement EfficiencyModule wide lever that impacts refinement from R20 (copper) to R4 (ammo)
15.) Toolkit Refinement EfficiencyModule wide lever that impacts refinement from R20 (iron) to R4 (toolkits)
16.) Combat Ammo BreakerModule wide lever that directly impacts the nominal cost of firing a weapon
17.) Fuel Combat BreakerModule wide lever that directly impacts the nominal cost of combat.
18.) HP Scaling FactorModule wide lever that scales health bars
19.) Crafting UGM MultiplierSB Upgrading Materials Crafting Main Breaker
20.) SB Upgrading BreakerSB upgrade curve exponent
21.) SB Upgrading MultiplierSB upgrade multiplier
22.) Fuel Impulse BreakerModule wide lever that directly impacts the nominal cost of impulse movement
23.) XP Emissions BaseNull
24.) XP Emissions Main BreakerNull
25.) XP Emissions Action MultiplierNull
26.) XP Emissions Combat DiscounterNull
27.) LP Emissions BaseUse this to scale the amount of time it takes to achieve a store tier
28.) LP Emissions Main BreakerUse this to increase the difficulty to obtain the higher LP store tiers
29.) LP Emissions Action MultiplierUse this to scale the amount of LP dropped from a particular LP action
Impact to Star Atlas Economy
First and foremost, the combination of lever actuations means that there is a near limitless number of economic combinations that could result of these levers. More importantly, the Star Atlas economic teams has in place a robust set of tools should they need to loosen or tighten economic loops.
Second, data runners have uncovered assigned numerical values to these levers. These numerical values provide insight into economic costs. For example, based on assigned values, Food is used twice as much for Combat than it is for Mining. However these numerical values are subject to change in the future as a direct result from public play testing.
Lastly, these tools how a comprehensive approach to managing the Star Atlas economy. Certain tools like the Resource Scalar provide sweeping global changes to the economy. Contrast that with more precise tools like resource refinement efficiency that only effect one particular mechanism. Combined, the entire lever set highlights how the Economics team can either rapidly or gradually make changes to the overall state of the Star Atlas Economy.
