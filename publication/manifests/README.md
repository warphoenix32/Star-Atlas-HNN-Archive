# Publication Manifests

This directory contains internal publication ledgers governed by
[`../PUBLICATION-CONTRACT.md`](../PUBLICATION-CONTRACT.md).

`publication-manifest.json` is intentionally empty during Phase 3. Publication
entries are added only after the Phase 4 knowledge-consolidation gate produces
reviewed dossiers and Phase 5 authorizes an initial portfolio.

Only entries with state `PUBLISHED` may appear in the default public Library
build.
