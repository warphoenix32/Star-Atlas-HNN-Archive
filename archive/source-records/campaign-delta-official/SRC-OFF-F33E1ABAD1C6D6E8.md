## Preserved Official Text

The C4 PTR settings menu gives you access to account options, character options, support tools, keypair import/export, and basic game settings.
This article gives a quick overview of the options available from the settings menu.
Note: C4 PTR is still in development, so some settings and menu options may change over time.
Where to find the settings menu
The settings menu is available from the player menu in the top-right corner of the screen.
From there, you can access several account and game options.
Account Settings
The Account Settings area shows account-related information, including:
Wallet balances
Daily check-in streak
You can check in once per 24 hours.
View Character Info
The View Character Info option allows you to access character-related account actions.
From this area, you can deposit or withdraw:
ATLAS
SOL
ZINK
Change Character
The Change Character option allows you to switch characters or create alternate characters.
This works similarly to character selection in other MMO-style games, where you can create and play more than one account or character.
Help & Support
The Help & Support option gives you another way to access support resources.
From there, you can reach reporting tools, forms, and support information.
Use this option if you need to report a bug, submit feedback, request a feature, or create a support ticket.
Disconnect Wallet
The Disconnect Wallet option signs your wallet out of the game.
Important for PTR: If you disconnect your wallet during PTR, you may need your keypair to get back into your saved game.
Make sure you understand the import/export keypair options before disconnecting.
Import Keypair
The Import Keypair option allows you to restore a saved PTR game using a keypair file.
Use this if you previously exported your keypair and need to access the same PTR profile again.
Export Keypair
The Export Keypair option allows you to save your PTR game as a keypair file.
You can use this file later to import your saved PTR profile again or access it from another device.
Because C4 PTR uses test assets and test wallet information, keeping your exported keypair safe is important if you want to keep access to the same PTR profile.
Command Settings
The Command Settings option controls basic game settings.
This includes basic audio and visual settings for the game.
Summary
The C4 PTR settings menu lets you manage important account and game options, including character information, support access, wallet disconnection, keypair import/export, and command settings.
Before disconnecting your wallet or switching devices, make sure you have exported your keypair if you want to keep access to the same PTR profile.
