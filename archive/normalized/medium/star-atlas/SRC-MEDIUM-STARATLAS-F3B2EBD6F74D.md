Welcome back, explorers! We’re excited to share the latest patch notes for February’s update. Before diving into the new enhancements, let’s quickly revisit the highlights from last December.

Last December’s Highlights

As 2024 came to a close, we launched the Star Atlas Unreal Engine 5 Private Preseason — our most significant update yet. It introduced new game modes, enhancements, and multiplayer features, making the Galia Expanse more accessible and enjoyable for our players.

Here’s a glance at the major updates:

Crewcards V0 Integration

We introduced a crew system to the Unreal Engine 5 client, enabling players to interact with their crew members for the first time. Though character arts are still under development, this initial step offers a sneak peek at the unique, customized experiences awaiting in Galia Expanse.

Deathmatch Modes in Paizul’s Arena

This new exciting map allows players to engage in quick, intense three-way team and free-for-all battles in public lobbies, using unique species, accessories, and weapons from Star Atlas.

Multiplayer Racing Tracks

We’ve added two thrilling maps on Planet Ioki’s sands for self-hosted racing lobbies. These high-speed tracks are designed to enhance our community’s competitive spirit.

New Single-Player Test Modes

Players can now explore two new environments in the Galia Expanse universe:

- ONI Faction Central Space Station : Navigate the expansive capital of the ONI Consortium and discover its district-by-district intricacies as it grows to be an economic metropolis of the Star Atlas metaverse
- Project Galia : This experimental mode invites players to fly through space, exploring sectors and discovering planets, previewing the future of Star Atlas.
Fleet Command Test Mode

The Unreal Engine 5 iteration of our blockchain SAGE game, Fleet Command, offers a real-time view of SAGE fleets across the galaxy, with feature parity expected later this year.


## What’s New in This Patch

General Updates

This patch adds a series of new features, which we will break down by topic. First, let’s look at the general updates that affect many game modes.

- Species Specifics Stats: The system now supports species-specific HP (health points) and Movement Speed. All characters currently equip the same shield.
- Kill Animation: New crosshair animations on hit-and-kill.
- Ships HUD: New Ship HUD notification when failing to switch to hover mode in microgravity.
- Hit animation: New damage direction indicators in Player HUD
- Death VFX: New death, kills, hit sounds.
- Waypoint Markers: Waypoint markers now have screen edge persistence.
- Spectator: Improved camera behavior on player death.
- Bots Spawn points: Bots in Gunplay Trainer no longer spawn randomly but at the furthest spawn point from existing players.
- Bot Racing: Remove Racing bots from time trials until they are more performant/competitive.
- Environment Refactoring: Refactor Openable Objects (doors, ramps, elevators) for better multiplayer support and easier interactions.
- Grenade Refactoring: Refactor grenades for better performance during multiplayer latency.
- Grenade Effects: New grenade explosion VFX.
- Ragdolls Animation: Animation update on ragdolls for better visuals.
- Ship Steering Animation: Polish pass on movement and steering animations for all XXS and XS ships.
- Art and Physics Enhancement: Various art, collision, and performance improvements across all Maps.
- Collision: Various art/collision/camera issues on spawnable ships.
- Optimization of The UI: Optimization pass on most UI screens/elements.
Wallet UX Enhancements

We’ve addressed several issues to improve the blockchain wallet experience:

- No Wallet Login: Logging into your wallet is now optional for Unreal Engine 5 game modes, simplifying onboarding for those new to blockchain.
- Mnemonic Recovery : Issues preventing mnemonic recovery have been resolved.
- Password Recognition : Fixed password recognition problems.
- Wallet Creation : Streamlined the wallet creation process. Players won’t be prompted to create a new wallet every time they log in.
- Auto Login : Enhanced auto-login for quicker access to previously used wallets.
Fleet Command Visualization Enhancements

We’ve made significant progress in the Fleet Command visualization game mode, improving ship displays UI/UX, and adding quality-of-life features for better gameplay. The improvements include:

- Optimized Ship VFX : thrusters, scanning, mining, warping.
- Updated Cataclysm hero asset: New art for the area on the center of the map, where the secrets of the Cataclysm lay hidden.
- Map enhancement : Updated lighting and art in galaxy and sector views.
- Celestial Bodies Art: Planet shader improvements.
- User Interface and User Experience Quality of Life: Updated UI screens with more data and improved UX.
- New Camera System: Augmentation of the user experience while exploring the map.
- Modularity: Decoupled various asset classes for better interface.
Private & Friend Lobbies

Players can now create private or friend-only lobbies for exclusive multiplayer experiences in Unreal Engine 5. The new updates incorporate:

- Private Gameplay: Add support for Private, Friends Only P2P session
- Multiplayer Sessions ID: Add support for joining P2P sessions by Session ID
- Deathmatch Ranking: Add Rank screen at the end of a Deathmatch.
- Minimap updates: Deathmatch minimap no longer shows all players as friendly
- Optimization: General networking optimizations for characters and ships
- Fleet Controls: Enable Power Triangle and Ship Boost in Multiplayer.
- Mounting ships: Clean up replication visuals when mounting ships
Optimizations, Polish, and Bug Fixes

This patch also introduces optimizations and bug fixes, improving performance and gameplay as we prepare for the public release later this year, such as:

- Restore the ability to link a wallet using 12-word pneumonic.
- Restore the ability to post Photos to discord from UE Client.
- Fix the issue where previously used wallets would not unlock given the correct password.
- Fix the issue where the player would not respawn correctly in Galia.
- Fix Pickups Invisible on Track B.
- Fix the issue where the HVT Crystal minimap marker would persist as a static gray icon.
- Fix the issue where race pickup sounds were loud and didn’t adhere to the volume settings.
- Fix the issue where the player can lose the ability to shoot after throwing a grenade.
- Fix issues where the ship can drift during a racing countdown.
- Fix Racing results screen “new record” and rank indicators.
- Fix the issue where the character would not enter the correct animation state when running and shooting automatic weapons.
- Fix the issue where the holstered weapon would appear on the hip when ADSing a sighted weapon after swapping.
- Fix the issue where the player could sometimes respawn in T-pose.
- Fix the issue where a free look in certain camera angles would not restore the default camera position when released.
- Bots can now correctly aim at species of shorter stature.
- Remove unused Ship model text from racing leaderboards
- Remove the change team option in the pause menu during a death match.
- Fix sound attenuation issues in certain corners of the Paizul map.
- Added Compakt Hero proxy mesh when viewing at a distance.
- The Play Now menu button works when the selected mode is Volant Exp.
- Fix an issue where grenade explosion could cause damage through walls/obstacles.
