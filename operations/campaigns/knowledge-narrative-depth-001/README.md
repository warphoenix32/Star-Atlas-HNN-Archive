# Knowledge Narrative Depth Campaign 001

This campaign converts evidence already merged on `main` into deeper, source-qualified institutional history. It does not rewrite archive evidence, graph facts, or publication outputs.

The campaign preserves proposal, vote, passage, authorization, payment, implementation, release, testing, completion, and independent verification as separate states. Council tracker fields remain attributed operational assessments. Community sources remain attributed and cannot displace stronger primary evidence.

PR #17 was already merged before this campaign began. PR #19, the separate Medium ingestion campaign, remains unmerged and is explicitly excluded from the evidence base.

Run:

```text
python build_campaign.py
python validate_campaign.py
```

Human semantic review is required before merge, especially for the R3 treasury, award, periodization, Town Hall, and Atlas Brew pages.
