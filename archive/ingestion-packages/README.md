# Ingestion Packages

This directory contains structured JSON extractions grouped by campaign, plus schema-versioned packages. Wave 1 campaign extractions retain their original bytes beneath each campaign’s `extractions/` directory.

Current campaign groups: Alpha (Aephia), Bravo (Intergalactic Herald), Charlie (HNN), and Delta (official sources). See [campaign summaries](../campaign-summaries/README.md) for coverage and failures.
