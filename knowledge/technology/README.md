# Technology Knowledge

Star Atlas combines game products with Solana programs, game-specific configuration, APIs, public repositories, Unreal Engine tooling, and test infrastructure. This domain explains those systems without turning documentation into proof of deployment or source code into proof of production use.

## Evidence rule

The technical collection keeps six questions separate: Was a system documented? Was source released? Was it deployed? Which product build used it? Was operation observed? Was the claim independently verified? A strong answer to one question does not silently answer the others.
